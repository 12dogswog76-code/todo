🗂️ Как это работает

Это простой список дел. Все данные хранятся в твоём браузере — никто, кроме тебя, их не видит. Регистрация не нужна.

Основы

  • Напиши задачу и нажми Enter — готово. Категория, этап, дедлайн и повтор — по желанию.
  • Клик по задаче — редактировать текст, категорию, этап, дедлайн и чек-лист подпунктов.
  • Дата + время = таймер обратного отсчёта (время без даты = сегодня). Просроченное подсветится красным.
  • ✓ — выполнено (опустится вниз через 2,5 часа, через неделю уедет в архив) · 📌 — закрепить · ● — цвет · ⠿ — перетащить · ▲▼ — порядок · ✕ — удалить.
  • 🔁 Повторяющаяся задача после выполнения сама создастся на следующий срок (каждый день / неделю / месяц).
  • ☑ Чек-лист: открой задачу кликом, в поле «+ подпункт» жми Enter — подпункты с галочками появятся прямо в задаче, со счётчиком «☑ 2/5».
  • 🔍 Поиск ищет по названиям, этапам и чек-листам. Фильтр «🗃 архив» — выполненные старше недели.
  • 🎨 Чип «по цвету» в фильтрах группирует задачи по цветам. Цвет задаче задаётся кнопкой ●.
  • 🏷 Свои категории: кнопка «✎ категории» рядом с чипами — задай название, выбери эмодзи из палитры, удаляй ненужные. Категории синхронизируются между устройствами.
  • 📊 «Статистика» внизу — сколько выполнено сегодня / за неделю / за месяц и серия дней подряд 🔥.
  • 🪟 «Поверх окон» внизу (Chrome на компе) — мини-список, который висит поверх всех программ: отмечай и добавляй задачи, не переключаясь.
  • Ссылку просто вставь в текст — она станет кликабельной.

☁ Синхронизация между устройствами

Чтобы список был одинаковым на телефоне, компе и ноуте, нужно бесплатное облако-хранилище. Мы используем jsonbin.io:

  1. Зарегистрируйся на jsonbin.io (нужна только почта, карта не нужна).
  2. В кабинете открой API Keys и скопируй Master Key.
  3. Здесь внизу нажми «☁ облако», вставь Master Key → «Подключить». Это делается один раз, только на первом устройстве.
  4. Нажми «Скопировать код подключения» и перешли его себе (например, в Telegram «Избранное»).
  5. На остальных устройствах: «☁ облако» → вставь этот код → «Подключить».

⚠️ Важно: Master Key вставляется только один раз на первом устройстве. На все остальные — только код подключения, иначе создастся новое пустое хранилище. Код подключения — это «ключ» от твоих задач, не делись им с чужими.

Дальше всё само: изменения улетают в облако через пару секунд, другие устройства подтягивают их при открытии и каждые 3 минуты. Точка возле заголовка: зелёная — всё ок, красная — нет связи (данные не теряются).

🔔 Уведомления

Кнопка «уведомления» внизу — браузер предупредит за 30 минут до дедлайна и в момент истечения. Работает, пока вкладка с сайтом открыта (можно в фоне).

📱 Как приложение

Сайт можно установить: в Chrome на компе — значок установки в адресной строке, на Android — меню ⋮ → «Установить приложение». Появится иконка, открывается в своём окне, работает даже без интернета.

Без облака: экспорт / импорт

Не хочешь заводить облако — переноси данные вручную: «⇪ экспорт» → «Скопировать код» на одном устройстве, «⇩ импорт» → вставить код → «Импортировать» на другом. Тем же экспортом удобно делать резервную копию в файл.
